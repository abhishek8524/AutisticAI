## Inspiration
SenseMap is a modern, accessible web application designed to help autistic and sensory-sensitive individuals discover calm, comfortable public spaces such as cafés, libraries, parks, and community areas. The platform uses an interactive map interface, sensory ratings, and AI-generated insights to help users plan visits that align with their personal comfort levels. This design prioritizes calm, clarity, and emotional comfort, ensuring that every interaction is predictable, low-stimulation, and intuitive.
## What it does
The idea for SenseMap came from the difficulty many sensory-sensitive individuals face when navigating public spaces. Loud environments, bright lighting, crowds, and sudden stimuli can make even routine outings stressful. We was inspired to create a tool that reduces uncertainty and increases confidence, allowing users to explore safely, make informed choices, and enjoy their surroundings.

Additionally, we drew inspiration from modern, minimalist digital products like Linear, Notion, and Apple’s Human Interface Guidelines to ensure the app feels professional, polished, and trustworthy.

Throughout this project, we deepened my understanding of:
1. Autism-friendly UX design: Implementing soft color palettes, generous spacing, low-contrast transitions, and predictable layouts to reduce cognitive load.
2. Accessibility standards: Designing with keyboard navigation, screen reader support, and motion-reduction options in mind.
3. Interactive mapping UX: Balancing complex information, like noise levels, lighting, crowd density, and sensory triggers without overwhelming the user.
4. AI integration for UX: Summarizing user reviews into actionable sensory insights using AI tools while keeping the interface calm and understandable.
5. Design system creation: Building reusable components such as map pins, location cards, filter sliders, and ranking panels with consistent visual language.
These lessons reinforced how thoughtful UX can empower users, not just provide information.
## How we built it
The project was built in several stages:
1. Research & Planning: Conducted surveys and reviewed accessibility and autism-friendly design guidelines. Compiled sensory factors affecting public spaces: noise, lighting, crowding, and common triggers.
2. Wireframing: Designed low-fidelity sketches of the main screens: Landing Page, Map Explorer, Location Detail, Review Submission, Sensory Rankings, and User Profile.
3. Prototyping: Built mid- to high-fidelity prototypes using soft color palettes and rounded components. 
4. Interaction Design: Filters update results instantly without overwhelming animations. Profile sliders provide live recommendations based on user sensitivities.
5. Testing & Iteration: Conducted usability testing with potential users and adjusted layouts, color contrasts, and animation speed. Fine-tuned AI summary panels to display concise sensory insights.
## Challenges we ran into
1. Balancing information density and simplicity: Map screens needed to show multiple sensory metrics without overwhelming the user.
2. Designing for emotional comfort: Ensuring that colors, animations, and layout did not overstimulate users while still feeling modern and professional.
3. Incorporating AI insights: Presenting AI-analyzed sensory reviews in a calm, digestible way without creating cognitive overload.
4. Accessibility compliance: Meeting WCAG standards while maintaining a visually appealing, modern UI.
## What we learned
SenseMap taught us that design is as much about empathy as functionality. Every choice, from color to layout, from slider speed to iconography impacts how safe and comfortable a user feels. By focusing on accessibility and sensory comfort, I was able to create a platform that not only informs but empowers users to confidently explore the world.
## What's next for SenseMap
Next, SenseMap aims to expand to mobile platforms, enhance personalization with AI-driven recommendations, and grow a community-driven database of sensory-friendly spaces. Future updates will include more detailed sensory data, accessibility improvements, real-time insights, and gentle engagement features, helping users explore the world confidently, safely, and comfortably.
